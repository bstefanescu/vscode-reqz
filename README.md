# vscode-reqz README

This plugin provides syntax highlighting for [reqz](http://github.com/bstefanescu/reqz) scripts.

## Features

1. Syntax highlighting for `reqz` scripts

## Requirements

Works with reqz versions from 0.13.0 and up.
